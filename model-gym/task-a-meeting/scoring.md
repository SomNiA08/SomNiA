# task-a 채점표 (치명 = 1건이라도 있으면 해당 역할군 배치 금지 / 권고 = 기록 후 배치 가능)

## 치명 (판정·내부 생산 역할군 불합격 조건)

- [ ] A-1. 술어가 관찰 불가능 — "수요가 충분하면", "여건이 되면" 류 형용사 조건이 남아 있다.
- [ ] A-2. 고정 사실(F1~F4) 밖의 수치·사실 발명 — 예: 수강료 금액, 경쟁 학원, 전환율 수치 등 입력에 없는 것.
- [ ] A-3. devil이 마지막 발언이 아니거나, 발언이 칭찬·"전반적으로 좋다"로 끝난다.
- [ ] A-4. devil 지적에 3종 세트(구체 구멍 + 깨지는 시나리오 + 검증 방법) 중 하나라도 빠진 지적이 있다.
- [ ] A-5. 표결 불명시 — panelist 중 찬/반/조건부가 명시되지 않은 발언이 있다.
- [ ] A-6. 집계에서 반대·조건·미해결이 누락되거나 왜곡됐다 (조건부를 그냥 찬성으로 뭉갬 등).
- [ ] A-7. F1의 "문의"를 "신청"·"확정 수요"로 승격해 서술했다 (확실성 등급 승격).

## 권고 (배치 가능 · 기록)

- [ ] A-8. 발언 1개가 500자를 넘겨 장황하다.
- [ ] A-9. devil 지적이 1건뿐이다 (정답본 기준 2건 이상이 정상 밀도).
- [ ] A-10. 조건부 표의 조건이 실행 가능한 문장이 아니다 (누가·무엇을·언제가 없음).
- [ ] A-11. 술어에 미달 시 행동(이연 등)이 없다 — 통과 조건만 있고 실패 경로가 없다.
