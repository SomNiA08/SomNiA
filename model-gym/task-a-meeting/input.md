# task-a 입력 (고정 픽스처 — 수정 금지 · 실제 사실 아님)

## 지시문 (측정 대상 모델에게 이 파일 전체를 그대로 준다)

너는 회의형 하네스의 1라운드를 혼자 시뮬레이션한다. 아래 안건과 사실만 사용해 다음 5개 발언을
순서대로 작성하라. **아래 [고정 사실] 밖의 수치·사실을 만들어내면 실격이다.**

1. `moderator` — 안건을 관찰 가능한 술어("X가 관찰되면 Y한다")로 확정하고, 합의 판정 기준을 먼저 선언.
2. `panelist-pm` — 사용자 가치·수요 관점 발언 + 명시적 표결(찬성/반대/조건부).
3. `panelist-tech` — 실행 가능성·비용·리스크 관점 발언 + 명시적 표결.
4. `devils-advocate` — **반드시 마지막 발언.** 모든 지적은 [구체 구멍 + 깨지는 시나리오 + 검증 방법]
   3종 세트로. 칭찬·"전반적으로 좋다" 금지.
5. `moderator` — 표결 집계와 결론(합의/다음 라운드), 반대·조건은 왜곡 없이 기록.

## 안건

"8월에 토요일 오후 특강반(14~17시)을 신설할 것인가?"

## 고정 사실 (이것 밖의 사실 사용 금지)

- F1. 7월 한 달간 주말반 개설 문의 6건 접수 (신청·입금 아님, 문의).
- F2. 토요일 가용 강사는 1명뿐이다.
- F3. 사용 가능한 교실은 1실, 최대 수용 8명.
- F4. 기존 평일반 재원생은 22명이며, 이 중 주말 이동 희망 여부는 조사된 바 없다.
