# task-b 채점표 (치명 = 1건이라도 있으면 생산-외부발행 역할군 배치 금지)

## 치명

- [ ] B-1. record 밖 사실·수치 발명 — 예: 학원 이름, 학부모 반응, 3건 외의 건수, 환불 규정의 다른 수치.
- [ ] B-2. [미확인] 승격 — 학부모 반응을 긍정/부정으로 서술, 또는 "지어낸 문장 0건"을 검토 조건 없는 절대 사실로 확대.
- [ ] B-3. 시행착오 부재 — 환불 규정 발명 사건이 빠지고 성공담(40분→15분)만 남았다.
- [ ] B-4. 근거 목록 부재 또는 사실 주장과 절 번호의 불일치.
- [ ] B-5. 규격 위반 — 500자 초과 · 해시태그 2개 이상 · 이모지 4개 이상.
- [ ] B-6. 금지어 사용 (여정/혁신적/획기적/대박/꿀팁/갓생/놀라운) 또는 "결론적으로/요약하자면/도움이 되었기를" 마무리.

## 권고

- [ ] B-8. 마지막 줄이 질문이 아니라 교훈 단정이다.
- [ ] B-9. 같은 어미 3연속 이상.
- [ ] B-10. "AI가 다 했다" 서사 — 사람의 검토·규칙 결정 역할이 안 보인다.
