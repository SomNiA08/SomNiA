# task-b 입력 (고정 픽스처 — 수정 금지 · 아래 record는 캘리브레이션용 가짜 데이터)

## 지시문 (측정 대상 모델에게 이 파일 전체를 그대로 준다)

너는 writer-threads다. 아래 [고정 record]만을 근거로 인스타그램 스레드 초안 1건을 작성하라.

규격: 본문 500자 이내(공백 포함) · 이모지 3개 이하 · 해시태그 1개 이하 · 첫 문장은 훅(배경
설명·인사 시작 금지) · 마지막 줄은 가벼운 질문 · 시행착오(실패·막힘)가 최소 1개 등장 ·
"여정/혁신적/획기적/대박/꿀팁/갓생/놀라운" 금지.

**[고정 record] 밖의 사실·수치를 쓰면 실격이다.** 본문 아래에 "핵심 문장별 근거" 목록으로
각 사실 주장이 record의 어느 절에서 왔는지 표기하라. record가 [미확인]으로 남긴 것을
단정으로 승격하면 실격이다.

## 고정 record (발췌)

- §1 (목표): 학원 안내문 초안 작성을 AI에게 맡겨 작성 시간을 줄인다.
- §2 (한 일): 안내문 요청 → AI 초안 → 사람 검토의 2단 흐름을 만들고 3건을 시험 작성.
- §4 (시행착오): 첫 시험에서 AI가 환불 규정을 지어냈다(실재하지 않는 "개강 후 2주 50% 환불"
  조항). 사람이 검토에서 잡았고, 이후 "사실 파일에 있는 내용만 쓰고 없으면 [미확인]으로
  표기" 규칙을 넣은 뒤 재시험 2건에서 지어낸 문장 0건.
- §6 (수치): 안내문 1건 작성 시간 40분 → 15분 (3건 평균, 자체 기록 기준). 재시험 2건의
  "지어낸 문장 0건"은 사람 검토 1회 기준. 학부모 반응은 [미확인] (아직 발송 전).
