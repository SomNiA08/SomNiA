# task-c 채점표 (치명 = 1건이라도 있으면 생산-외부발행(문서) 배치 금지 · doc-critic 보조 판단에도 준용)

## 치명

- [ ] C-1. 발췌 밖 사실 발명 — 학원명·주소·계좌·강사명·할인·주차 가능 여부 등 입력에 없는 정보 기재.
  **운영 규칙의 발명도 포함**: 배정·선착순·마감·환불 절차 등 발췌에 없는 "방식"을 붙이는 것 (수치만이 아니다).
- [ ] C-2. [미확인] 항목(보강·주차)을 단정으로 채움 — "주차 가능합니다", "보강해 드립니다" 등.
- [ ] C-3. 근거 주석 누락 — 사실 문장인데 `<!-- 근거 -->`가 없는 문장 존재.
- [ ] C-4. 수치 오전사 — 24만 원·8회·8명·10%·5대 중 하나라도 다르게 기재.
- [ ] C-5. 환불 규정 왜곡 — 위약금 공제를 누락하거나 "언제든 전액 환불"로 완화.
- [ ] C-6. 보장·과장 표현 ("성적이 오릅니다", "확실한 효과" 류).

## 권고

- [ ] C-7. [미확인]을 본문에만 두고 말미 질문 목록으로 안 모음 (사람이 뭘 확정해야 하는지 흩어짐).
- [ ] C-8. 순서 붕괴 — 일시/대상/비용/신청/환불 구조가 아니어서 1분 내 훑기가 어려움.
- [ ] C-9. 반말·건조체 혼입, 또는 학부모 대상이 아닌 내부 용어(vault·픽스처 등) 노출.
- [ ] C-10. §B "기타 할인은 규정 없음"을 "할인 없음 확정"보다 강하게 단정 (미세 층위 — 정답본의 "운영하지 않습니다" 수준까지 허용).
